Point Multipliers (บาท/จุด)

วิธีตั้งค่า Google Sheets

  1. สร้าง Google Sheet ใหม่
  2. ไปที่ Extensions → Apps Script
  3. ลบ code เดิมแล้ววาง code จากไฟล์ google_apps_script.js
  4. กด Deploy → New deployment
  5. เลือก Type: Web app
  6. ตั้ง Execute as: Me
  7. ตั้ง Who has access: Anyone
  8. กด Deploy แล้ว Copy URL มาวางในช่องด้านบน
